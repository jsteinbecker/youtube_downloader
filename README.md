# youtube_downloader

## Add Links

Create cue of links to download audio within `links.txt`, one line per link.

Ex:

```txt
www.youtube.com/link-1
www.youtube.com/link-2
...
```
## Execution

- Execute `script.py`
- Completed links can be left in `links.txt` --- link copied to `completed`, where the YouTube-to-mp3 function will not be executed duplicatively if you run the script back to back.
